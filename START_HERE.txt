⚠️ 注意事項

1. 圖片文件需要你自己準備
   放置位置：themes/wave-harmonics/assets/images/
   需要的圖片：
   - ff_1.png, ff_2.png, ff_3.png, ff_4.png
   - fo_1.png, fo_2.png, fo_3.png, fo_4.png

2. 建議使用本地伺服器（不是必須，但更好）
   python -m http.server 8000
   或使用 VS Code Live Server

3. 支援的瀏覽器
   ✅ Chrome/Edge（推薦）
   ✅ Firefox
   ✅ Safari
   ⚠️ IE 不支援

   - 破音懲罰
   - 錯題重現
   - 音效系統

✅ 視覺化工具
   - 題目編輯器（含 AI 生成）

   - 審題系統（完整驗證）

❓ 常見問題

Q: 圖片無法顯示？
A: 檢查 themes/wave-harmonics/assets/images/ 
   確保所有圖片都存在

Q: 遊戲無法啟動？
A: 打開瀏覽器 Console (F12) 檢查錯誤訊息
   或參考 QUICK_TEST.md

Q: 如何創建新主題？
A: 參考 FILE_MANIFEST.md 的「場景 4」

Q: 如何修改題目？
A: 使用 question-editor-v2.html
   或直接編輯 questions-v2.json
